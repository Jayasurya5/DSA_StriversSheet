# Striver's A2Z DSA Sheet - File Structure

This file contains the complete folder and file structure for organizing practice problems.

## Structure Convention
- `source/` - Contains C++ implementation files (.cpp)
- `documentation/` - Contains markdown documentation files
- Each section numbered with two digits (01_, 02_, etc.)
- Each subsection numbered with two digits
- Each file numbered with two digits within its folder
